# Tic-Tac-Toe

A tic-tac-toe GUI for `Node.js`.

---
## Requirements

|              |         |
|--------------|---------|
| `Node.js`    | `^14.0` |
| `npm`        | `^6.0`  |
| `typescript` | `^4.0`  |


---
## Project setup

```bash
npm install
```

---
## Run the game

Run:
```bash
npm run play # build and run script
```

Then follow instructions.  
Cells must be input based on their index from `0` to `9`.
